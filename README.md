## Project Overview
This repository contains a literary analysis of George Orwell's *1984*, focusing on the sociolinguistic themes of Newspeak and state control. This project was completed as part of a Practical Assignment (PA) to demonstrate proficiency in FOSS (Free and Open Source Software) tools.

## Tools Used
* **Overleaf (LaTeX):** Used for document formatting and professional typesetting.
* **ZoteroBib:** Used for managing academic references in BibLaTeX format.
* **GitHub:** Used for version control and digital hosting of the project.

## Files in this Repository
1. **main.pdf**: The final compiled version of the literary analysis.
2. **source.zip**: The complete LaTeX source code, including the .tex and .bib files.
